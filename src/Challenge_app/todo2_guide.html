🎛️ 操作ガイド

各フォームとボタンの使い方

スクリーンショットに表示されているすべての入力欄とボタンを説明します。

① 入力フォーム
タイトル タスク名を入力（必須）。空白のまま追加しようとするとエラーになります。
期限日 「日付を選択」ボタンでカレンダーが開きます。「クリア」で期限なし（ハイフン「-」表示）に戻せます。今日以前の日付は設定不可。
優先度 高・中・低から選択（デフォルトは「中」）。
メモ 補足を自由に入力（任意）。
① アクションボタン
タスク追加

フォームの内容で新しいタスクを追加します。編集中は「更新」に変わります。

選択を編集

期限切れ・未完了リストで1件だけチェックした状態で押すと、その内容がフォームに読み込まれます。

選択を一括完了 選択を削除

チェックした複数タスクをまとめて完了 / 削除します。

全選択 選択解除 編集解除

「編集解除」は編集モードをキャンセルしてフォームをリセットします。

① 並び替えコンボボックス
入力順 タスクを追加した順番。デフォルトの並び順です。
日付順 期限日が早い順に並びます。期限なし「-」は末尾に表示。
優先度順 高 → 中 → 低 の順。同優先度内は日付順で安定ソートされます。
④ 完了エリアのボタン
未完了に戻す

完了エリアでチェックしたタスクを未完了状態に戻します。

完了選択を削除

完了エリアでチェックしたタスクを削除します（元には戻せません）。

リストのタスク表示フォーマット

各リストエリアのチェックボックスには以下の形式でテキストが表示されます。タイトルは最大15文字で切り詰め、全角換算で幅揃えされます。

 1. 先月の請求書処理　　　 | 期限: 2026-06-01 | 優先度: 高 | メモ: 経理へ提出
 2. レポートを書く　　　　 | 期限: 2026-07-15 | 優先度: 高 | メモ: 参考文献を先に集める
 3. 買い物リスト作成　　　 | 期限: -　　　　 | 優先度: 低 | メモ:

期限切れエリアの行は赤字で表示されます。

🚀 起動方法

exe 版と Python 版

配布 exe ファイルを受け取った場合は Python 不要。ソースコードがある場合は Python で直接実行できます。

A

exe 版（Python インストール不要）

配布された ToDoApp.exe を好きな場所に置いてダブルクリックするだけで起動します。

📁 デスクトップ
    📄 ToDoApp.exe ← ダブルクリックで起動
    📄 todo_data2.json ← 起動後に自動作成
⚠️ 「Windows によって PC が保護されました」と表示された場合は 「詳細情報」→「実行」 をクリックしてください。
B

Python 版（ソースコードを直接実行）

Python 3 がインストールされていれば、追加ライブラリのインストールは不要です。

# todo2.py があるフォルダに移動してから実行
python todo2.py
💡 Python 3 系であれば動作します。追加でインストールが必要なライブラリはありません（tkinter は Python 標準付属）。
✨ 機能一覧

7つの機能

スクリーンショットに写っているすべての機能です。

📅

カレンダーで期限日を設定

「日付を選択」ボタンでポップアップカレンダーが開きます。月の移動もできます。「クリア」で期限なしに戻せます。

☑️

チェックで一括完了

未完了・期限切れエリアで複数チェックして「選択を一括完了」を押すとまとめて完了にできます。

🔴

期限切れを自動検出・赤表示

今日より前の期限を持つ未完了タスクは「期限切れ」エリアに自動移動し、赤字で警告表示されます。

✏️

タスクの編集

1件だけチェックして「選択を編集」を押すとフォームに読み込まれます。変更後「更新」で上書き保存します。

↩️

完了を未完了に戻す

完了エリアで「未完了に戻す」を押すと選択したタスクを未完了状態に戻せます。

🔃

3種類の並び替え

入力順・日付順・優先度順の3モードを切り替えられます。並び替えはすべてのリストエリアに適用されます。

💾

自動保存

タスクの追加・更新・完了・削除のたびに todo_data2.json へ自動保存されます。アプリを閉じてもデータは残ります。

💾 データ保存

保存ファイルは1つだけ

exe・todo2.py と同じフォルダに自動作成されます。

📁 同じフォルダに作られるファイル
ToDoApp.exe
アプリ本体（exe 版）。ダブルクリックで起動します。
todo_data2.json
全タスクの保存ファイル。起動時に自動作成され、操作のたびに自動更新されます。
💡 バックアップ：todo_data2.json をコピーするだけで全データをバックアップできます。JSON 形式なのでテキストエディターでも中身を確認できます。
🔄 移行：ToDoApp.exe と todo_data2.json を一緒にコピーすれば、別のフォルダや別の PC でもそのままデータを引き継いで使えます。
❓ FAQ

よくある質問

気になることはこちらで解決！

Q. Python をインストールしないと使えませんか？

A. exe 版（ToDoApp.exe）であれば Python は不要です。ダブルクリックだけで起動できます。

Q. セキュリティソフトが「危険」と警告しました。

A. Python で作成した exe は誤検知されることがあります。配布元を信頼できる場合は「詳細情報 → 実行」を選択してください。

Q. データはインターネットに送られますか？

A. 送られません。 完全オフラインで動作します。すべてのデータは PC 内の todo_data2.json にのみ保存されます。

Q. 削除したタスクを元に戻せますか？

A. 削除後は元に戻せません。事前に todo_data2.json をバックアップしておくことをおすすめします。なお「完了」状態のタスクは「未完了に戻す」で復元できます。

Q. Mac や Linux でも使えますか？

A. exe 版は Windows（10 / 11）専用です。Mac / Linux で使う場合は Python 3 をインストールして python todo2.py で直接実行してください。
