🙋 このアプリについて

タスクをまとめて
スッキリ管理

締め切り・担当者・メモ・コメントを一元管理。期限前には通知してくれます。

  • 📚
    学校のレポートや仕事の作業をリスト化して管理したい
  • 👥
    複数人で担当を分けてタスクを進めたい
  • ⏰
    期限を忘れないようにリマインダーをセットしたい
完全オフライン動作
🚀 スタートガイド

4ステップで起動

難しい設定は不要！Python があれば今すぐ使えます。

1

Python をインストールする

まだ Python が入っていない場合は python.org からダウンロード＆インストール。バージョン 3.x であれば動作します。

2

コマンドプロンプト（またはターミナル）を開く

Windows の場合は「スタートメニュー → コマンドプロンプト」または「PowerShell」を開きます。

3

以下のコマンドを入力して Enter を押す

# task_manager.py があるフォルダに移動してから実行
python task_manager.py
💡
ヒント：追加でインストールが必要なライブラリはありません。Python 標準の機能だけで動きます！
4

ウィンドウが開いたら完成！

アプリのウィンドウが表示されればすぐに使い始められます。データは自動的に保存されます。

➕

タスクの追加・編集・削除

「＋ 新規タスク」ボタンで新しいタスクを作成。タイトル・カテゴリ・担当者・期限などを入力できます。

🏷️

カテゴリ & タグ

「仕事」「個人」「学習」などカテゴリで分類。さらにタグ（例：#急ぎ）でグループ化できます。

👥

担当者の割り当て

チームメンバーを登録して各タスクに担当者を設定。誰が何をやるか一目でわかります。

🌲

サブタスク

大きなタスクを小さなタスクに分割できます。親タスクには進捗（例: 2/5）が表示されます。

⏰

リマインダー

期限の「15分前」「1時間前」「1日前」などを設定すると、ポップアップで通知してくれます。

💬

コメント

タスクにコメントを書き込めます。誰がいつ書いたか自動記録されるので進捗共有に便利。

🔔

通知センター

タスクの作成・更新・担当変更などの履歴を一覧表示。未読件数がバッジで表示されます。

🔍

絞り込み検索

キーワード・カテゴリ・担当者・状態などで表示するタスクを絞り込めます。

💾 データ管理

データはどこに保存される？

アプリを終了してもデータは自動保存。バックアップも簡単です。

ファイル何が入っている？
data/tm_tasks.json タスク一覧（タイトル・状態・期限・コメントなど）
data/tm_categories.json カテゴリ一覧
data/tm_assignees.json 担当者一覧
data/tm_notifications.json 通知履歴
data/tm_counter.json タスク ID カウンター
💡
バックアップ方法：data/ フォルダごとコピーするだけでデータをバックアップできます。JSON ファイルなのでテキストエディターでも中身を確認できます！
❓ よくある質問

FAQ

気になることはこちらで解決！

Q. 追加でインストールが必要なものはありますか？

A. いいえ、Python 3 があれば OK です。追加ライブラリのインストールは不要です。

Q. データはインターネットに送られますか？

A. いいえ、すべてのデータはあなたのパソコン内にのみ保存されます。完全オフラインで動作します。

Q. 削除したタスクは元に戻せますか？

A. 削除前に確認ダイアログが表示されますが、削除後は元に戻せません。大切なデータは data/tm_tasks.db をバックアップしておきましょう。

Q. 何人まで担当者を登録できますか？

A. 上限はありません。「👥 担当者管理」ボタンから何人でも追加できます。

Q. スマートフォンでも使えますか？

A. このアプリはパソコン（Windows / macOS / Linux）専用です。スマートフォンには対応していません。
