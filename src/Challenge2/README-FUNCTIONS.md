# 関数学習プログラム - 初心者向けガイド

このフォルダにある関数学習用プログラムは、段階的に関数の使い方を学べるように設計されています。

## プログラム一覧（推奨学習順）

### 1️⃣ **func-basic.py** - 最もシンプルな関数
- **学ぶこと**: 関数の定義と呼び出し
- **特徴**: パラメータなし、戻り値なし
- **実行**: `python func-basic.py`

### 2️⃣ **func-params.py** - パラメータを渡す
- **学ぶこと**: パラメータの使い方、複数パラメータ
- **特徴**: 異なる値を渡して関数の結果を変える
- **実行**: `python func-params.py`

### 3️⃣ **func-return.py** - 戻り値を返す
- **学ぶこと**: `return`文で値を返す、計算結果を再利用
- **特徴**: 関数の結果を変数に保存したり、直接表示したり
- **実行**: `python func-return.py`

### 4️⃣ **func-bmi-calc.py** - 実践例：BMI計算機
- **学ぶこと**: 複数の関数を組み合わせる、条件分岐との組み合わせ
- **特徴**: 実際に使える計算機
- **実行**: `python func-bmi-calc.py`

### 5️⃣ **func-list.py** - リスト処理と関数
- **学ぶこと**: デフォルト引数、リストを扱う関数
- **特徴**: `sum()`や`len()`のような標準関数を自作する感覚を学ぶ
- **実行**: `python func-list.py`

### 6️⃣ **func-grade-system.py** - 総合例：成績管理システム
- **学ぶこと**: 複数の関数を組み合わせて実用的なシステムを作る
- **特徴**: 辞書処理、複数の処理を関数に分ける
- **実行**: `python func-grade-system.py`

## 関数の基本ルール

```python
# 関数の定義
def 関数名(パラメータ1, パラメータ2):
    """説明（ドキュメンテーション）"""
    # 処理
    return 戻り値  # 省略可能

# 関数の呼び出し
結果 = 関数名(引数1, 引数2)
```

## 学習ポイント

| ポイント | 説明 |
|--------|------|
| **定義** | `def`で関数を定義 |
| **パラメータ** | 関数に渡す値、関数の括弧の中 |
| **戻り値** | `return`で関数の結果を返す |
| **呼び出し** | 関数名()で実行 |
| **デフォルト引数** | パラメータに `=` で初期値を設定 |

## 実践のコツ

1. **簡単なものから始める** - func-basic.py → func-params.py の順番を守る
2. **コードを読んで理解する** - 実行前にコードをじっくり読む
3. **実行してみる** - 実際に動かして動作を確認
4. **アレンジしてみる** - パラメータを変更して試してみる
5. **自分で書いてみる** - 既存コードを参考に、自分で関数を作成してみる

## よくある質問

**Q: `return`がない関数は何が返される？**  
A: `None` という値が返されます

**Q: パラメータがない関数はどう定義する？**  
A: `def 関数名():` と括弧を空にします

**Q: 複数の値を返したい時は？**  
A: タプルやリストで複数の値を返します: `return a, b, c`

**Q: 関数の中で関数を呼ぶことはできる？**  
A: はい！プログラムが複雑になると関数を組み合わせることが重要です
（func-grade-system.py を参照）
