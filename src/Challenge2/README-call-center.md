# コールセンター勤怠管理システム - 使い方ガイド

## 🎯 プログラム概要

このプログラムは、コールセンターの勤怠管理と給料計算を自動化します。

**対応する機能：**
- ✅ 30名の従業員管理
- ✅ 早番（9:00-18:00）と遅番（12:00-21:00）の管理
- ✅ 20:00以降の遅番1.25倍割増計算
- ✅ 日給計算
- ✅ 月間給料集計
- ✅ CSV出力

## 📊 勤務シフト

| シフト | 勤務時間 | 時間 | 割増対象 |
|--------|---------|------|---------|
| 早番 | 9:00-18:00 | 9時間 | なし |
| 遅番 | 12:00-21:00 | 9時間 | 20:00-21:00（1時間）× 1.25倍 |

## 💰 給料計算例

**早番の日給：**
- 9時間 × ¥1,200/時 = **¥10,800**

**遅番の日給：**
- 基本給：8時間 × ¥1,200/時 = ¥9,600
- 割増給：1時間 × ¥1,200/時 × 1.25 = ¥1,500
- **合計：¥11,100**

## 🚀 実行方法

```bash
python call-center-attendance.py
```

## 📋 出力内容

プログラムを実行すると以下の情報が表示されます：

1. **勤怠管理表** - 全従業員の情報
2. **集計サマリー** - 総人数、シフト別人数、総給料
3. **CSVファイル** - `勤怠管理表.csv` を作成

## ⚙️ カスタマイズ方法

### 時給を変更する場合

```python
BASE_HOURLY_WAGE = 1500  # 1200 → 1500に変更
```

### 割増率を変更する場合

```python
PREMIUM_RATE = 1.5  # 1.25 → 1.5に変更
```

### シフト時間を変更する場合

```python
EARLY_SHIFT_START = 8    # 9 → 8に変更
EARLY_SHIFT_END = 17     # 18 → 17に変更
LATE_SHIFT_START = 13    # 12 → 13に変更
LATE_SHIFT_END = 22      # 21 → 22に変更
```

### 割増対象時間を変更する場合

```python
PREMIUM_START_TIME = 19  # 20 → 19に変更
```

### 従業員情報を手動で設定する場合

自動生成の代わりに、以下のように直接設定できます：

```python
employees = [
    {"id": "E001", "name": "田中太郎", "shift": "早番"},
    {"id": "E002", "name": "佐藤花子", "shift": "遅番"},
    {"id": "E003", "name": "鈴木次郎", "shift": "早番"},
    # ... 以下30名まで
]
```

## 🔧 プログラムで使用されている関数

| 関数 | 説明 |
|------|------|
| `calculate_work_hours()` | シフト別の勤務時間を計算 |
| `calculate_premium_hours()` | 割増対象の時間を計算 |
| `calculate_regular_hours()` | 通常時給の対象時間を計算 |
| `calculate_daily_wage()` | 1日の給料を計算 |
| `display_employee_info()` | 従業員1名の情報を表示 |
| `display_all_employees()` | 全従業員の情報を表示 |
| `calculate_total_payroll()` | 全従業員の総給料を計算 |
| `count_shift_distribution()` | シフト別人数をカウント |
| `display_summary()` | サマリーを表示 |
| `export_to_csv()` | CSVファイルにエクスポート |

## 🐛 トラブルシューティング

**Q: CSVファイルが文字化けする**  
A: エクセルで開く時に「データの取得」→「テキストファイル」で「UTF-8」を選択してください

**Q: 従業員数を変更したい**  
A: 以下を変更してください：
```python
employees = [
    {"id": f"E{i+1:03d}", "name": f"従業員{i+1}", "shift": "早番" if i % 2 == 0 else "遅番"}
    for i in range(50)  # 30 → 50に変更
]
```

**Q: 毎日の出勤/欠勤を管理したい**  
A: `employees` の辞書に `"worked_days"` フィールドを追加して管理できます
